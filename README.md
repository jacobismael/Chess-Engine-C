# Chess Engine in C

Chess Engine written in C.

## Setup

Build the executable:
```bash
./setup.sh
```

or:
```bash
mkdir build
cd build
make
```

- - -

## Execution:

Visual Mode (unfinished):
```bash
./chess
```

Terminal Mode
```bash
cd build
./chess <Name> <Location> <Output File>
```
